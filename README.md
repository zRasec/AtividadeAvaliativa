# Aplicativo de Saudação (Android)

Aplicativo Android simples desenvolvido como atividade avaliativa em sala de aula.

## Descrição

O aplicativo permite que o usuário digite seu nome em um campo de texto.  
Ao pressionar o botão, o aplicativo exibe uma mensagem de saudação personalizada na tela.

Exemplo de funcionamento:

Digite o nome:
Julio

Resultado exibido:
Olá Julio

## Funcionalidades

- Entrada de texto utilizando **EditText**
- Botão para executar a ação
- Exibição da mensagem utilizando **TextView**
- Manipulação de eventos com **OnClickListener**

## Tecnologias utilizadas

- Java
- Android SDK
- Android Studio

## Objetivo da atividade

Praticar conceitos básicos de desenvolvimento Android:

- Manipulação de componentes da interface
- Captura de dados digitados pelo usuário
- Eventos de clique em botões
- Atualização dinâmica de texto na interface
